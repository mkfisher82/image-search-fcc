Image Search Abstraction Layer Microservice

To do an image search:

Add the suffix '/api/imagesearch/searchterm' to the end of this url.

For example: https://pacific-thicket-49812.herokuapp.com/api/imagesearch/freecodecamp

To do an image search but miss a number of the first results:

Add the suffix '/api/imagesearch/searchterm?offset=#' to the end of this url where '#' is the first result shown.

For example: https://pacific-thicket-49812.herokuapp.com/api/imagesearch/freecodecamp?offset=5

To see a list of the latest search requests:

Add the suffix '/api/latestsearches' to the end of this url.

For example: https://pacific-thicket-49812.herokuapp.com/api/latestsearches
